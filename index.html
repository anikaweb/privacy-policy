Política de Privacidad — InkFlow Studio

Esta Política de Privacidad describe cómo se recopila, utiliza y protege la información de los usuarios de la aplicación InkFlow Studio. Al utilizar la aplicación, aceptas las prácticas descritas en esta política.

1. Información que recopilamos

1.1 Información proporcionada por el usuario

InkFlow Studio puede permitir que el usuario introduzca datos dentro de la aplicación, los cuales pueden almacenarse en Firebase Realtime Database o Firestore. Estos datos pueden incluir:

  • Información básica introducida por el usuario según las funciones de la app
  • Preferencias o configuraciones dentro de la aplicación

InkFlow Studio no solicita información personal sensible como direcciones físicas, documentos oficiales o datos bancarios.

1.2 Información recopilada automáticamente

La aplicación recopila automáticamente ciertos datos técnicos necesarios para su funcionamiento:

  • Modelo del dispositivo, sistema operativo, idioma
  • Identificadores anónimos
  • Datos de uso y rendimiento
  • Información sobre fallos y errores

2. Servicios de terceros utilizados

2.1 Google AdMob (Anuncios)

InkFlow Studio muestra anuncios mediante Google AdMob, que puede recopilar:

  • Identificadores de publicidad
  • Datos de interacción con anuncios
  • Información del dispositivo

Política de Google: https://policies.google.com/privacy

2.2 Firebase (Base de datos y servicios)

InkFlow Studio utiliza Firebase para:

  • Almacenar datos del usuario
  • Sincronizar información
  • Gestionar autenticación (si aplica)

Política de Firebase: https://firebase.google.com/support/privacy

2.3 Google Analytics / Firebase Analytics

La aplicación utiliza Google Analytics para analizar el uso de la aplicación. Este servicio recopila datos anónimos como:

  • Pantallas visitadas
  • Tiempo de uso
  • Eventos dentro de la app
  • Tipo de dispositivo

Política de Google Analytics: https://policies.google.com/privacy

3. Permisos utilizados

InkFlow Studio puede solicitar permisos como:

  • Acceso a Internet
  • Acceso a almacenamiento (si la app lo requiere)
  • Otros permisos estrictamente necesarios para su funcionamiento

4. Uso de la información

La información recopilada se utiliza para:

  • Garantizar el correcto funcionamiento de la aplicación
  • Mostrar anuncios personalizados o no personalizados
  • Analizar el rendimiento y mejorar la experiencia del usuario
  • Detectar errores y optimizar funciones
  • Sincronizar datos mediante Firebase

5. Compartición de información

InkFlow Studio no comparte información personal identificable con terceros. Los únicos datos compartidos son los necesarios para los servicios mencionados (AdMob, Firebase, Analytics), bajo sus propias políticas.

6. Seguridad

Aplicamos medidas razonables para proteger la información almacenada en Firebase. Sin embargo, ningún sistema es completamente seguro.

7. Privacidad de menores

InkFlow Studio no está dirigida a menores de 13 años y no recopilamos conscientemente información personal de niños.

8. Cambios en esta política

Podemos actualizar esta Política de Privacidad ocasionalmente. La versión más reciente estará siempre disponible en esta misma URL.
